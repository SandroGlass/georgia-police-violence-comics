# Police Ultraviolence in Georgia - Interactive Comics

An interactive comic storytelling experience documenting cases of police violence during the 2024 pro-EU protests in Georgia.

## Overview

This project presents real stories of victims of police violence during the November 2024 protests in Tbilisi, Georgia, when thousands took to the streets following the government's decision to halt EU accession talks.

## Featured Stories

### Giorgi Gamgebeli's Story
Photographer was beaten by masked forces on November 30th in Tbilisi while documenting protests. He suffered a broken leg and destroyed camera.

### Tornike Chelidze's Story  
Poet and teacher was arrested and beaten by special forces on November 30th in Tbilisi during pro-European protests, sentenced to 7 days in prison.

## Technical Features

- **Interactive Comic Panels**: Smooth GSAP animations and transitions
- **Responsive Design**: Works on all devices and screen sizes
- **Touch Navigation**: Swipe gestures for mobile devices
- **Keyboard Controls**: Arrow keys for navigation
- **Progressive Loading**: Optimized for performance
- **Accessibility**: Proper ARIA labels and semantic markup

## Technologies Used

- HTML5 with semantic markup
- CSS3 with advanced animations and responsive design
- JavaScript (ES6+) for interactivity
- GSAP (GreenSock) for smooth animations
- Modern web APIs for touch and keyboard events

## Browser Support

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge
- Mobile browsers (iOS Safari, Android Chrome)

## Usage

1. Open `index.html` in a web browser
2. Navigate through the main menu
3. Select a story to begin
4. Use touch gestures, arrow keys, or navigation buttons to progress through scenes

## Navigation Controls

- **Touch**: Swipe left/right to navigate between scenes
- **Keyboard**: Use arrow keys (← →) to navigate, Escape to return to menu
- **Buttons**: Use the navigation bar at the bottom of scenes

## Development

The project uses vanilla JavaScript and modern CSS features. No build process is required - simply open the HTML file in a browser.

This project aims to document and preserve testimonies of victims of police violence during peaceful protests in Georgia. The stories presented are based on real events and testimonies collected by human rights organizations.
